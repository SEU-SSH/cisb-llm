**Null pointer check incorrectly deleted for static pointer**

```
int main()
{
    static char* ptr = (char*)1;
    --ptr;
    if(ptr){
        return 1;
    }
    return 0;
}

Compiled with -O0, it returns 0
Compiled with -O1, it returns 1
Compiled with -O1 -fno-delete-null-pointer-checks, it returns 0
```
